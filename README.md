# zepto-book-shop
Live: https://zepto-book-shop.vercel.app

# Project Run Documentation in Local
-   Clone the repo.
-   Open terminal
    -   Run **yarn** or **npm install**.
    -   Run **yarn dev** or **npm run dev** to see the result in browser.
